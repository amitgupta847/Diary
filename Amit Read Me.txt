How to run?
1. Latest NODEJS and TypeScript is installed.
2. Open command prompt from the folder where package.json is placed.
3. run "npm install" to install all Angular 2 packages.
4. run "typings install".
5. Now Open solution in Visual Studio 2015
6. At "Task Runner Explorer", right click on "default" task and run to create libs folder,
 and to transpiles TypeScript files to JavaScript.
7. Now run app to see Angular 2 running on ASP.NET MVC 5.
